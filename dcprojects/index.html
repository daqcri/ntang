Short Description

NADEEF is a data cleaning platform, which abstracts a set of programming APIs such that heterogeneous data quality rules (e.g., conditional functional dependencies, editing rules and sherlock rules) can be implememted in a homogeneous way. NADEEF also provides a runtime execution engine to detect data violations. We also provide a graphical HTML5 user interface to help users navigate and understand their data in order to improve the rule quality.

FALCON is an interactive, deterministic, and declarative data cleaning system.

KATARA is a trusted data cleaning system powered by knowledge bases and crowdsourcing.

A long way to go . . .

Without a shadow of a doubt, data cleaning has played an important part in the history of data management and data analytics. Possessing high quality data has been proven to be crucial for businesses to do data driven decision making, especially within the information age and the era of big data. However, in general, data cleaning problems are very hard, and heuristic algorithms are widely used, which fall short of any accurcy guranttees. Many researches have been conducted by using relaible external sources, such as master data, users, knowledge bases, confidence values. They shed considerable light on effective data cleaning. In the real world, there is still a long way to go.
